Resources

You can iterate all resource chunks available in the PSD file by invoking the method parseResources().

This will scan the resources chunk and extract all resources and create an object for each added to the array. The object has properties:

res.id   - id of resource (see Adobe® file specification for details)
res.name - name of resource if any
res.size - size of resource in bytes
res.pos  - start position of data for this resource in the buffer

Now you can use the info object to scan through all resources:

var res1 = psd.info.resources[0];

Or instead, you can look up a resource by ID using the method findResource(id). Invoking this method will also automatically invoke parseResource() if not called previously.

Note that for indexed color mode file the parseResources() are automatically invoked in order to find the transparency index (if any).
